== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as regular WordPress plugin.
2. Go your Plugins setting via WordPress Dashboard and activate it.
3. After activating the plugin just copy shortcode from shortcode page and paste anywhere in your post page or widgets sections to display post Grid.<br /> 
<br />

<strong>Using FTP</strong>
1. Extract post-grid-free.zip to your computer.
2. Upload the post-grid-free directory to your wp-content/plugins directory.
3. After activating the plugin just copy shortcode from shortcode page and paste anywhere in your post page or widgets sections to display post Grid.

<br />
<strong>How to use short-code?</strong><br />
When plugin active & all setup done please copy shortcode from shortcode page and paste anywhere in your post page or widgets sections to display post Grid.
